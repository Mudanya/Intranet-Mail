The system needs to be run on a server.
Install it on the webserver from which users can access it through the LAN.


Or


Test it first through local server that is apache tomcat.
Install the apache tomcat and ensure to configure Catalina home variable.
Import the database provided in mysql platform of your choice.
Copy the servlet files and put in the web folder of local server.
Call localhost:8080//i-Mail/signup.jsp from any browser when the 
server is on and you can proceed as  illustrated in the diagrams in the documentation.
